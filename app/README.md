Prerequisites

  g++ for c++11,
  
  nodejs,
  
  node packages:
      express
      formidable
      fs
      path

Installing

  use 'npm install "package"' for node package

Running

  exec the script 'startServer' (prerequisites: firefox)

  or

  run the command 'nodejs app.js' and access pages from the localhost with port 3000.
